# Breast Cancer Prediction System

A production-ready healthcare web application that utilizes a deep learning Neural Network to classify breast tumors (Benign vs. Malignant) based on 30 cell nucleus features. 

## Tech Stack
* **Backend:** Python, Flask
* **Machine Learning:** TensorFlow/Keras, Scikit-Learn, NumPy, Joblib
* **Frontend:** HTML5, CSS3 (Vanilla/Custom), Vanilla JavaScript
